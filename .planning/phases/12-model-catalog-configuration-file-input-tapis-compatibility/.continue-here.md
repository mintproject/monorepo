## Critical Anti-Patterns (do NOT repeat)
- [ARRAY-WRAP-JUNCTION-SCALAR]: `out[snakeToCamel(colName)] = [val]` looks consistent with v1.8.0 object-property array wrap, but junction columns are scalars. JS truthy on `[false]` makes UI lie. → Always emit junction column hoist as scalar.
- [LAZY-REFETCH-DROPS-JUNCTION-COLUMN]: `ModelCatalogResource.setResources` refetches each row via `resourceApi.get(id)` and stores in `_loadedResources`. Render and `getResources()` read from there, NOT the inline payload. Any junction-only field is lost. → Subclass must capture inline overlay and merge on `requestUpdate`.
- [PR-RUN-LOOKS-LIKE-SKIP]: GitHub Actions fires push + pull_request runs per commit. `if: github.event_name == 'push'` correctly skips build on PR run. Don't panic when one of two runs shows "build-and-push skipped" — check the push run.
