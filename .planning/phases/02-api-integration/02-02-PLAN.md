  <action>
1. Create `src/hasura/client.ts`:
   - Import `GraphQLClient` from `graphql-request`
   - Read `HASURA_GRAPHQL_URL` from env (default: `http://localhost:8080/v1/graphql`)
   - Read `HASURA_ADMIN_SECRET` from env (optional)
   - Export a function `getClient(headers?: Record<string, string>): GraphQLClient` that creates a client with the URL and merges admin secret header if set (`X-Hasura-Admin-Secret`)
   - Export a `readClient` instance pre-configured with admin secret for read operations
   - Export a function `writeClient(bearerToken: string): GraphQLClient` that creates a client forwarding the user's `Authorization: Bearer <token>` header

2. Create `codegen.ts` (graphql-codegen config):
   ```typescript
   import type { CodegenConfig } from '@graphql-codegen/cli'
   const config: CodegenConfig = {
     schema: [{
       [process.env.HASURA_GRAPHQL_URL || 'http://localhost:8080/v1/graphql']: {
         headers: {
           'X-Hasura-Admin-Secret': process.env.HASURA_ADMIN_SECRET || 'myadminsecretkey'
         }
       }
     }],
     documents: 'src/hasura/**/*.graphql',
     generates: {
       'src/generated/sdk.ts': {
         plugins: [
           'typescript',
           'typescript-operations',
           'typescript-graphql-request'
         ]
       }
     }
   }
   export default config
   ```

3. Create `src/app.ts`:
   - Import Fastify, @fastify/cors, @fastify/swagger, @fastify/swagger-ui
   - Export an async `buildApp()` function that:
     a. Creates a Fastify instance with `logger: { level: process.env.LOG_LEVEL || 'info' }` using pino's built-in structured JSON logging
     b. Registers @fastify/cors with `origin: true`
     c. Registers @fastify/swagger (to serve the spec)
     d. Registers @fastify/swagger-ui at `/v2.0.0/docs`
     e. Adds a GET `/health` route that calls Hasura with a simple introspection query (`{ __typename }`) and returns `{ status: 'ok', hasura: 'connected' }` on success or `{ status: 'error', hasura: 'unreachable' }` with 503 on failure
     f. Do NOT register openapi-glue yet (that comes in plan 03 when service handlers exist) -- just add a placeholder comment
     g. Returns the Fastify instance

4. Create `src/index.ts`:
   - Import `buildApp` from `./app`
   - Call `buildApp()`, then `app.listen({ port: Number(process.env.PORT || 3000), host: '0.0.0.0' })`
   - Log startup message with port
   - Handle SIGTERM/SIGINT for graceful shutdown (`app.close()`)
  </action>
